## Description

The **mobidisc** project is designed for drawing multiloops in surfaces and computing their mobidiscs and unicorn annuli.

## Installation

To set up the project, ensure you have Python installed on your system. You can install the required dependencies using the `requirements.txt` file provided in the project directory. Run the following command in your terminal:

```bash
pip install -r requirements.txt
```

## Usage

After installing the dependencies, you can run the main application using:

```bash
python src/main.py
```
